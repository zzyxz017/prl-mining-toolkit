# PRL Mining Toolkit - Windows Desktop App

8-tab tkinter application for tracking PRL mining profitability.

## Build
\`\`\`
pyinstaller --clean --onefile --windowed --name PRL_mining_toolkit mining_toolkit_win.py
\`\`\`

## Data
Stored in ~/.hermes/mining_toolkit/

## Tabs
Profitability, Breakeven, Daily Mining, Sales, Transfers, Drag Analysis, Trends, Data
